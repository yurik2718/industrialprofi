# Промпт для генерации изображений (инфографика)

Картинки-инфографики к урокам через **Nano Banana**. Бриф каждой картинки уже написан — это
`alt` плейсхолдера в уроке: `![<alt>](TODO-...png)`.

## Как пользоваться

1. Возьми `alt` плейсхолдера → это содержимое картинки.
2. Вставь его в `КОНТЕНТ` шаблона ниже; блок `СТИЛЬ` оставь без изменений.
3. Сгенерируй, бегло проверь подписи/числа. Криво — попроси модель поправить точечно.

## Фиксированные решения

- **Фон:** тёмный с ровной точечной сеткой. Чтобы точки были ровные — дай модели **готовую
  подложку на вход** и проси рисовать поверх (подложку сделай один раз, CSS ниже).
- **Шрифт:** Inter (заголовки — Inter Tight, жирные).
- **Язык:** подписи по-русски; термины и единицы латиницей (`Modbus`, `OPC UA`, `4–20 mA`,
  `AI/AO/DI/DO`, `%IW`, `RS-485`, `SCADA`, `SIL`); аббревиатуры `АСУ ТП`, `ПЛК`, `ПАЗ` — кириллицей.
- **Формат:** 4:3 по умолчанию; 16:9 — для широких потоков; 1:1 — для вертикальных схем.
- **Без водяного знака.** Изображения чистые — без `industrialprofi.com` и любых
  логотипов (осознанное решение, см. `CLAUDE.md`: водяной знак противоречит
  CC BY-SA, захламляет учебную деталь и не даёт SEO-пользы).

Подложка-фон (отрендери в CSS, сделай скриншот нужного размера, переиспользуй на всех картинках):

```css
background-color: #0A0A0A;
background-image: radial-gradient(#262626 1.5px, transparent 1.5px);
background-size: 28px 28px;
```

## Шаблон промпта

```
СТИЛЬ (не менять): высококачественная техническая инфографика, как от дизайнера, НЕ похожая на
AI. Плоский вектор на тёмном фоне #0A0A0A с ровной точечной сеткой; near-white #F5F5F5 тонкие
линии и текст, один синий акцент #3B82F6 на ключевом элементе. Тонкие иконки в стиле Tabler,
строгая сетка и выравнивание, чёткая иерархия (заголовок → схема → подписи), воздух. Весь текст
шрифтом Inter — рисуй чётко и без ошибок. Подписи по-русски, термины латиницей точно как даны.
Соотношение сторон <4:3 / 16:9 / 1:1>. Без водяного знака и логотипов.
Рисуй инфографику поверх приложенного фона, фон с точками не меняй.

КОНТЕНТ: <текст alt из урока — что изобразить, элементы, стрелки, выноски>.
Подписи буквально: "<заголовок>", "<подпись>", … (≤6–8 штук).

NEGATIVE: искажённый или выдуманный текст, фотореализм, 3D, глянец, градиенты, мусорные детали,
кривое выравнивание, неоновые/радужные цвета, клипарт, посторонние логотипы.
```

## Пример (урок про 4–20 мА, формат 16:9)

`СТИЛЬ` и `NEGATIVE` — как в шаблоне. `КОНТЕНТ`:

```
Двухпроводная токовая петля 4–20 mA слева направо: блок питания, датчик-передатчик, длинный
кабель, аналоговый вход ПЛК с резистором; тонкие стрелки «ток одинаков по всей петле».
Подписи буквально: "ТОКОВАЯ ПЕТЛЯ 4–20 mA", "Датчик", "ПЛК · вход AI", "24 V", "250 Ω",
"4 mA = 0%", "20 mA = 100%", "0 mA = обрыв".
```

## Чек-лист

- [ ] Текст верный и читаемый; кириллица целая, термины латиницей правильные, числа точны.
- [ ] Фон ровный (с подложки), точки не гуляют; тёмная база + синий акцент — «как приложение».
- [ ] Нет водяного знака и посторонних логотипов.
- [ ] Выглядит как дизайнерская инфографика, а не AI.

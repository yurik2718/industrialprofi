Практика для среднего уровня: выполни полную наладку токарного ЧПУ для новой детали по шагам — от чертежа до серийного пуска.
---
## Зачем этот чек-лист

Наладка токарного ЧПУ — это последовательность. Пропустить один шаг значит узнать об этом позже, в виде брака или аварии. Этот чек-лист — не инструкция «на всякий случай», а рабочий инструмент: используй его при каждой наладке новой детали, пока последовательность не войдёт в мышечную память.

Пять этапов, 20 шагов. Каждый шаг — конкретное действие с проверяемым результатом.

---

## Этап 1: Подготовка (шаги 1–4)

### Шаг 1. Получи чертёж и технологическую карту

Что проверить:
- [ ] Чертёж с допусками на все ключевые размеры
- [ ] Технологическая карта: какой инструмент, какие режимы резания (V, f, ap)
- [ ] Материал заготовки совпадает с чертежом
- [ ] Количество операций и порядок обработки

> [!ВАЖНО]
> Не начинай наладку без полного комплекта документов. «Помню с прошлого раза» — источник брака.

### Шаг 2. Проверь заготовку

Что проверить:
- [ ] Размеры заготовки: диаметр и длина соответствуют чертежу (достаточно припуска на обработку)
- [ ] Нет трещин, расслоений, посторонних включений
- [ ] Материал промаркирован или сопровождается сертификатом
- [ ] Количество заготовок совпадает с заданием

### Шаг 3. Выбери и подготовь инструмент

Что сделать:
- [ ] Инструмент соответствует технологической карте (тип, пластина, покрытие)
- [ ] Пластины не изношены (не сколоты, режущая кромка в норме)
- [ ] Вылет резца из резцедержателя — не более 1,5 высоты корпуса (иначе вибрация)
- [ ] Все болты резцедержателя чистые, резьба не сорвана

### Шаг 4. Проверь станок перед запуском

Что проверить (согласно Приказу № 887н):
- [ ] Уровень СОЖ и масла в гидравлике
- [ ] Чистота рабочей зоны — нет посторонних предметов
- [ ] Защитная дверь исправна, блокировка работает
- [ ] Аварийный стоп (E-Stop) нажимается и сбрасывается штатно
- [ ] Нет посторонних шумов при ручном вращении шпинделя (на малых оборотах)

---

## Этап 2: Механика (шаги 5–8)

### Шаг 5. Установи заготовку в патрон

- [ ] Очисти кулачки и посадочную поверхность от стружки
- [ ] Установи заготовку, обеспечив правильный вылет (не более 3 диаметров без подпора)
- [ ] Предварительно зажми руками, выровняй при необходимости
- [ ] Затяни ключом патрона — равномерно, с нужным моментом
- [ ] Убери ключ из патрона (ключ в патроне = угроза жизни)

### Шаг 6. Проверь биение заготовки

- [ ] Подведи индикатор часового типа к наружной поверхности
- [ ] Прокрути шпиндель вручную — зафиксируй биение
- [ ] Допустимое биение для черновой обработки: до 0,5 мм; для чистовой: до 0,05 мм
- [ ] Если больше — перезажми или скорректируй кулачками

> [!ПРИМЕР]
> Заготовка Ø55 мм, чистовая операция. Индикатор показывает биение 0,3 мм — слишком много. Ослабляем кулачки, поворачиваем заготовку на 90°, затягиваем снова. Биение стало 0,05 мм — допустимо.

### Шаг 7. Установи инструменты в резцедержатель

- [ ] Каждый инструмент установлен в нужную позицию (T01, T02, T03…)
- [ ] Вылет резца зафиксирован (отметь маркером или запиши), затяжка болтов
- [ ] Ось режущей кромки проходит через ось шпинделя (по высоте центров)
- [ ] Вылет фиксируешь: изменение вылета = необходима перепривязка

### Шаг 8. Проверь режущую кромку по высоте центров

- [ ] Верхушка резца должна совпадать с осью вращения шпинделя ±0,05 мм
- [ ] Проверь по центру задней бабки (хвост центра — нулевая линия)
- [ ] Если выше — ухудшается Ra, возможен «потолок»; ниже — хуже стойкость

---

## Этап 3: Привязка (шаги 9–12)

### Шаг 9. Привяжи все инструменты по X

Для каждого инструмента:
- [ ] Запусти шпиндель на малых оборотах
- [ ] Сделай проход по наружному диаметру (0,1–0,2 мм)
- [ ] Отведи по Z, не трогая X
- [ ] Измерь диаметр
- [ ] Запиши в OFFSET → GEOMETRY → T0X → X → MEASURE

### Шаг 10. Привяжи все инструменты по Z

Для каждого инструмента:
- [ ] Подведи к торцу заготовки, сделай подрез (0,05–0,1 мм)
- [ ] Не двигай Z
- [ ] OFFSET → GEOMETRY → T0X → Z → введи 0.0 → MEASURE

> [!ВАЖНО]
> Привязывай каждый инструмент отдельно, по очереди. Не копируй значения с одного инструмента на другой — у каждого своя геометрия.

### Шаг 11. Установи нуль детали G54

- [ ] Определи, где нуль детали по чертежу (обычно правый торец, Z=0)
- [ ] Торец уже обработан при привязке по Z — нуль установлен автоматически
- [ ] Проверь значения G54 в меню: X должен соответствовать смещению оси, Z — положению торца

### Шаг 12. Проверь все коррекции перед запуском

- [ ] Открой таблицу OFFSET, пройди по всем T0x
- [ ] Wear X и Wear Z = 0,0 (или с запланированным припуском +0,3–0,5)
- [ ] Geometry X и Z — ненулевые разумные значения (X: реальный диаметр, Z: не 0 и не 999)
- [ ] Радиус кромки (R) введён, если используется G41/G42

---

## Этап 4: Первый запуск (шаги 13–17)

### Шаг 13. Загрузи программу

- [ ] Название программы совпадает с техкартой
- [ ] Программа загружена из правильного источника (не устаревшая версия)
- [ ] Визуально проверь первые и последние 10 строк
- [ ] Проверь: M08 (СОЖ), M03/M04 (направление), G40 в конце

### Шаг 14. Воздушный рез (Dry Run)

- [ ] Подними Z-смещение в G54 на +100 мм
- [ ] Feed Override — 50%
- [ ] Запусти программу в AUTO, наблюдай за траекторией
- [ ] Нет неожиданных резких движений, нет G00 в опасной зоне
- [ ] Верни G54 Z в исходное положение

### Шаг 15. Первый рез в Single Block с припуском

- [ ] Включи Single Block
- [ ] Feed Override — 30%
- [ ] В Wear X профильного инструмента введи +0,5 (припуск)
- [ ] Запускай программу кадр за кадром, наблюдай за каждым движением
- [ ] Рука на E-Stop весь первый проход

> [!ОПАСНО]
> Первый запуск новой программы — только Single Block, только с рукой на аварийном стопе. Нет исключений. Опытный наладчик делает это так же, как начинающий — это не мерило опыта, это стандарт безопасности.

### Шаг 16. Замер первой детали

- [ ] Остановил программу после чистового прохода (или специально запрограммированной остановки M01)
- [ ] Замерил все ключевые размеры: диаметры, длины, торцы
- [ ] Записал результаты: что есть / что нужно / отклонение

### Шаг 17. Ввод коррекций и запуск чистового прохода

- [ ] Рассчитал коррекцию: Измеренное − Номинальное = Ошибка (со знаком «−» в Wear)
- [ ] Ввёл коррекции в Wear X (и Z если нужно)
- [ ] Запустил чистовой проход
- [ ] Замерил — попали в допуск?
- [ ] Если да: снял режим Single Block, Feed Override — 100%

---

## Этап 5: Серийный запуск (шаги 18–20)

### Шаг 18. Запусти серию

- [ ] Feed Override — 100%, Single Block выключен
- [ ] СОЖ течёт в зону резания
- [ ] Дверь закрыта
- [ ] Наблюдай за первыми 2–3 деталями: звук резания, стружка, Ra

### Шаг 19. Первая деталь серии — 100% контроль

- [ ] Измерь все размеры из чертежа — не только ключевые
- [ ] Ra — при необходимости профилометром
- [ ] Запиши в маршрутную карту: деталь № 1, все замеры, подпись наладчика

### Шаг 20. Выборочный контроль по ходу производства

- [ ] Каждые 5–10 деталей — замер ключевого размера
- [ ] При дрейфе размера (тренд) — коррекция на износ до выхода за допуск
- [ ] При смене пластины — перепривязка + контроль первой детали после смены
- [ ] Фиксируй результаты в карте контроля (если требуется техкартой)

## Задание

### Цель

Выполнить полную наладку токарного ЧПУ для учебной детали «Вал ступенчатый» и произвести партию из 3 деталей в допуске.

### Понадобится

- Токарный ЧПУ (или симулятор: CNC Simulator Pro — бесплатная пробная версия; VNCK Virtual Machine)
- Заготовка: сталь 45, Ø50 мм, длина 100 мм (или параметры под имеющийся станок)
- Резец проходной CNMG 120408 или аналог
- Штангенциркуль (±0,05 мм), микрометр 25–50 мм (±0,01 мм)
- Чертёж детали (выдаёт мастер или преподаватель; при работе в симуляторе — задание из программы)

### Шаги

1. Пройди все 20 шагов чек-листа по порядку — не пропуская.
2. На каждом шаге отмечай галочку и записывай результат (числа, не просто «ок»).
3. Запиши в журнал: значения привязки инструментов, нуль G54, результаты замеров первой детали.
4. Произведи 3 детали. Замерь каждую по ключевым размерам, запиши в карту.

> [!ОПАСНО]
> При работе на реальном станке: всегда закрывай дверь, всегда зажимай деталь ключом, никогда не оставляй ключ в патроне, держи руку на E-Stop при первом запуске.

### Что сдать

Запись в журнале с:
- Таблицей: шаг → действие → результат (20 строк)
- Значениями привязки инструмента (X и Z geometry)
- Результатами замеров трёх деталей с указанием допуска

### Самопроверка

- [ ] Все 3 детали в поле допуска по диаметру (согласно чертежу)?
- [ ] Нуль детали G54 установлен и проверен перед первым запуском?
- [ ] Первый рез выполнен в Single Block?
- [ ] Перепривязка выполнялась после каждой смены пластины?
- [ ] В журнале есть замеры — числа, а не «в норме»?

---

> [!ПРОВЕРЬ]
> 1. Ты дошёл до шага 15 и обнаружил, что при воздушном резе инструмент проходил очень близко к патрону. Что ты сделаешь перед первым реальным резом?
> 2. После 8-й детали диаметр стал 0,04 мм больше допуска. Пластину не меняли. Что это означает и что делаешь?
> 3. Почему первую деталь серии (шаг 19) нужно проверять по 100% параметров, а не только по ключевым размерам?

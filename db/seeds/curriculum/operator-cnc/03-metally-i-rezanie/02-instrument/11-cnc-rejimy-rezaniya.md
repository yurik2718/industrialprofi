## Ментальная модель: три рычага управления резанием

Режим резания — это три взаимосвязанных рычага. Потянешь один — меняются два других. Понять связь между ними важнее, чем запомнить конкретные цифры.

**Vc (скорость резания, м/мин)** — как быстро режущая кромка движется относительно детали. Влияет прежде всего на **стойкость инструмента**: слишком высокая Vc → пластина перегревается и сгорает за минуты; слишком низкая → наростообразование, плохая поверхность.

**f (подача, мм/об)** — сколько металла срезается за один оборот шпинделя. Влияет на **шероховатость поверхности** и **производительность**: высокая подача → быстрее снимаем металл, но поверхность грубее; низкая → медленно, зато Ra меньше.

**ap (глубина резания, мм)** — насколько глубоко врезается инструмент за один проход. Влияет на **нагрузку на инструмент и станок**: большая глубина → большие силы резания, вибрация, риск поломки; малая → нужно больше проходов.

> Запомни приоритет, в котором задают режимы: **сначала ap → затем f → в последнюю очередь Vc**. Глубина определяет, сколько надо снять; подача — с каким усилием; скорость подстраивается под стойкость инструмента.

## Формула для расчёта оборотов шпинделя

Станок не «знает» скорость резания напрямую — он управляется оборотами шпинделя (n, об/мин). Скорость резания зависит от оборотов и диаметра заготовки (или инструмента).

$$n = \frac{V_c \times 1000}{\pi \times D}$$

где:
- **n** — частота вращения шпинделя, об/мин
- **Vc** — скорость резания, м/мин (берётся из каталога пластины / справочника)
- **D** — диаметр заготовки (при точении) или инструмента (при фрезеровании/сверлении), мм
- **π ≈ 3,14159**

**Обратная формула** (что такое Vc при известных n и D):

$$V_c = \frac{\pi \times D \times n}{1000}$$

> [!СОВЕТ]
> Запомни упрощённую оценку: Vc ≈ D × n / 318. Это быстро считается в уме: диаметр 50 мм, 1000 об/мин → Vc ≈ 50 × 1000 / 318 ≈ 157 м/мин.

## Разбор примера — пошаговый расчёт режимов точения

> [!ПРИМЕР] Деталь Ø50 мм из стали 45, черновая обточка пластиной P25
>
> **Исходные данные:**
> - Материал: сталь 45, HB 200, состояние — горячекатаный прокат
> - Диаметр заготовки: D = 50 мм
> - Пластина: CNMG 120408, класс P25, покрытие TiAlN
> - Операция: черновая обточка (нужно снять припуск 5 мм на диаметр → 2,5 мм на сторону)
> - Требование к Ra: не нормируется (черновая)
>
> **Шаг 1. Выбираем глубину резания ap.**
> Припуск на сторону = 2,5 мм. Снимем за один проход → ap = 2,5 мм.
> Проверяем: для пластины CNMG 120408 максимальная рекомендованная ap ≈ 4–5 мм (длина РК = 12 мм, ap берём ≤ 2/3 длины). Подходит.
>
> **Шаг 2. Выбираем подачу f.**
> Черновая, Ra не важна → берём f = 0,25 мм/об (по каталогу Sandvik для класса P25, черновая).
> Проверяем по максимальной нагрузке: радиус вершины пластины r = 0,8 мм, f ≤ r (правило для чистовой), но для черновой f может быть выше. Ок.
>
> **Шаг 3. Выбираем Vc.**
> Из каталога для стали 45 (P25, TiAlN): Vc = 180–220 м/мин. Берём середину: Vc = 200 м/мин.
>
> **Шаг 4. Считаем обороты шпинделя.**
> n = (200 × 1000) / (3,14 × 50) = 200 000 / 157 ≈ **1274 об/мин**
> Округляем до стандартного ряда станка: ставим 1250 об/мин.
>
> **Шаг 5. Уточняем фактическую Vc при n = 1250 об/мин.**
> Vc_факт = (3,14 × 50 × 1250) / 1000 = 196 250 / 1000 ≈ 196 м/мин. Нормально, в допуске.
>
> **Шаг 6. Считаем скорость подачи (минутную подачу Vf) для управляющей программы.**
> Vf = f × n = 0,25 × 1250 = **312 мм/мин** — именно это значение пишется в G-коде (команда F312).
>
> **Итог режимов:** n = 1250 об/мин, F312 мм/мин, ap = 2,5 мм.
> При этих режимах ожидаемая стойкость пластины T ≈ 25–30 мин.

## Чёрновая vs чистовая обработка

Это не просто разные режимы — это разная **задача** и разная **логика** выбора параметров.

| Параметр             | Чёрновая обработка                  | Чистовая обработка                  |
|----------------------|-------------------------------------|-------------------------------------|
| **Цель**             | Максимальный съём металла           | Точный размер и чистая поверхность  |
| **ap**               | Большая (1–5 мм и более)            | Малая (0,1–0,5 мм)                 |
| **f**                | Высокая (0,2–0,5 мм/об)             | Малая (0,05–0,15 мм/об)            |
| **Vc**               | Средняя (главное — стойкость)        | Высокая (меньше нарост)            |
| **Пластина**         | Прочная кромка, большой рад. вершины (r = 0,8–1,6 мм) | Острая кромка, малый r = 0,2–0,4 мм |
| **Контроль**         | Не контролируют Ra                  | Обязательно Ra и размер            |

**Правило шероховатости** (теоретическая Ra для точения):

$$Ra \approx \frac{f^2}{8 \times r}$$

где r — радиус при вершине пластины. Пример: f = 0,1 мм/об, r = 0,4 мм → Ra ≈ 0,01²/(8 × 0,0004) = 0,0001/0,0032 = 0,03 мм = 30 мкм = Ra 3,2.

Эта формула даёт теоретический минимум. Реальная Ra всегда хуже из-за вибраций, состояния кромки и материала. Используй её как **нижнюю оценку**: если теоретическая Ra уже больше нужной — уменьшай f или увеличивай r.

## Стойкость инструмента (T) и износ пластины

Стойкость — это время работы пластины (в минутах) до того, как её надо повернуть или заменить. По ISO 3685 критерием износа считается:

- **VB ≤ 0,3 мм** — равномерный износ задней поверхности для чистовых операций
- **VB ≤ 0,6–0,8 мм** — для черновых операций

На практике: **смотри на качество поверхности и размер детали**. Как только деталь вышла за поле допуска или Ra ухудшилась — пластина отработала.

**Что сокращает стойкость (типовые ошибки):**

- Vc слишком высокая → пластина сгорает, кратковременный белый дым от зоны резания
- Vc слишком низкая → наростообразование → вырывание кромки → катастрофический износ
- Прерывистое резание без специальных пластин → ударные нагрузки скалывают кромку
- Недостаточная подача на нержавейке → трение по наклёпанному слою

> [!ВАЖНО]
> Режимы из справочника — отправная точка, а не приговор. После первого прохода смотри на стружку, температуру, качество поверхности. Если стружка синеет и дымит — снижай Vc на 10–15%. Если поверхность шершавая при правильной Ra — уменьши f или смени пластину.

> [!ОПАСНО]
> Никогда не устанавливай режимы «с потолка» на новом материале без проверки в каталоге. Завышенная Vc на нержавейке или закалённой стали приводит к мгновенному выходу пластины из строя и возможному травмированию осколками.

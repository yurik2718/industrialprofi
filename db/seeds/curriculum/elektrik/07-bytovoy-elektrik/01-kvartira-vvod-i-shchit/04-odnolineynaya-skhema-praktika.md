Составь грамотную однолинейную схему реальной квартиры — по ней другой электрик должен понять всю электроустановку без единого вопроса.
---
## Задание

## Цель

Составить полную однолинейную схему реальной (своей или учебной) квартиры со всеми группами, кабелями, аппаратами защиты и указанием нагрузок. Схема должна быть достаточно подробной, чтобы по ней можно было сделать монтаж без дополнительных вопросов.

## Понадобится

- План квартиры (нарисованный от руки — нормально; достаточно пропорциональной схемы с указанием комнат)
- Карандаш или ручка, линейка
- ПУЭ Таблица 1.3.4 (допустимые длительные токи для кабелей — открой онлайн)
- Калькулятор
- Опционально: QElectroTech (бесплатная программа для однолинейных схем, есть русский интерфейс) — скачать с qelectrotech.org

## Шаги

**1. Составь список нагрузок по комнатам**

Обойди квартиру и запиши все точки: освещение (сколько светильников), розетки (сколько, что подключается), специальные приборы (стиральная, плита, кондиционер). Если квартира учебная — используй типовой набор: 3-комнатная квартира, 65 м², стандартный набор техники.

**2. Сгруппируй нагрузки в цепи**

Принцип группировки:
- Отдельная цепь для каждого мощного потребителя (варочная панель, духовка, стиральная машина, посудомойка, каждый кондиционер)
- Розетки — по помещениям или зонам (кухня отдельно, каждая жилая комната)
- Освещение — по зонам (не обязательно по одной комнате, можно совместить прихожую с коридором)
- Ванная — обязательно отдельная группа (или отдельные на освещение и тёплый пол)

**3. Рассчитай ток каждой группы**

Для активной нагрузки (нагреватели, лампы, кухонная техника):

```
I = P / U = P / 220
```

Для нагрузки с двигателем (стиральная машина, насос):

```
I = P / (U × cosφ)   где cosφ ≈ 0.85 для бытовой техники с двигателем
```

Пример: стиральная машина 2000 Вт:
`I = 2000 / (220 × 0.85) = 10.7А` → автомат 16А (следующий стандартный номинал)

**4. Выбери сечение кабеля**

Из ПУЭ Таблица 1.3.4 (кабель ВВГнг-LS, прокладка в трубе/штробе):

| Сечение мм² | Допустимый ток | Что подходит |
|---|---|---|
| 1.5 | 15А | Освещение (автомат 10А) |
| 2.5 | 21А | Розетки, стиральная (автомат 16А) |
| 4 | 27А | Духовка (автомат 25А) |
| 6 | 34А | Варочная панель (автомат 32А) |

**Правило:** номинал автомата ≤ допустимый ток кабеля.

**5. Определи, где нужна дифзащита 30мА**

Обязательно:
- Все цепи ванной комнаты
- Стиральная машина, посудомойка → дифавтомат тип **A**
- Тёплый пол в ванной
- Кухонные розетки (по современным нормам)

Тип A (не AC) — для всего, что содержит электронику с выпрямителями (стиралки, посудомойки, кондиционеры с инвертором).

**6. Нарисуй схему**

Стандартные обозначения:
- Автомат: прямоугольник с диагональю, рядом номинал (например, «С16»)
- УЗО: прямоугольник с обозначением «△I=30мА»
- Дифавтомат: объединённый блок (автомат + УЗО)
- Кабель: линия с подписью «ВВГнг-LS 3×2.5»
- Счётчик: прямоугольник с кружком и «kWh»

Нарисуй сверху вниз: ввод → вводной автомат → счётчик → противопожарное УЗО → шины PE/N → группы.

**7. Проверь схему по чек-листу**

До сдачи пройди по каждому пункту самопроверки ниже.

## Что сдать

Фото или скан нарисованной схемы. Схема должна читаться — если нарисована от руки, используй линейку и пиши чётко. Приложи таблицу с расчётами токов (пункт 3).

## Самопроверка

- [ ] Для каждой группы указаны: марка и сечение кабеля, номинал и полюсность автомата, описание нагрузки
- [ ] Номинал каждого автомата ≤ допустимому длительному току кабеля (по ПУЭ 1.3.4)
- [ ] Ванная комната имеет дифзащиту 30мА на всех цепях
- [ ] Стиральная и посудомоечная машины — дифавтомат тип A (не AC)
- [ ] Варочная панель — отдельный кабель 3×6 и автомат 32А
- [ ] Духовой шкаф — отдельный кабель 3×4 и автомат 25А
- [ ] На схеме указан вводной кабель с маркой и сечением
- [ ] Есть противопожарное УЗО 100–300мА или дифзащита 30мА индивидуально на каждой опасной группе

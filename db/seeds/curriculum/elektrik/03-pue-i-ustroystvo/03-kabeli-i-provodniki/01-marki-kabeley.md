Маркировка кабеля — это зашифрованный паспорт: материал жилы, тип изоляции, оболочка и класс пожарной безопасности читаются прямо из аббревиатуры.
---
## Что ты поймёшь после урока

Ты сможешь расшифровать любую марку кабеля и объяснить, почему ВВГнг-LS обязателен в публичных зданиях, а алюминиевый кабель запрещён в жилых домах.

**Ментальная модель.** Маркировка кабеля — это этикетка продукта: читаешь слева направо, каждый блок символов означает один конструктивный элемент или класс характеристик.

---

## Как читать российскую маркировку кабеля

Маркировка строится по следующей логике (слева направо):

| Позиция | Обозначение | Значение |
|---------|------------|---------|
| Материал жилы | *(нет буквы)* = медь, **А** = алюминий | АВВГнг: алюминиевые жилы |
| Тип изоляции жил | **В** = ПВХ, **Р** = резина, **С** = сшитый полиэтилен (СПЭ) | |
| Тип оболочки | **В** = ПВХ, **К** = резина, **Б** = бронированный, **нг** = нераспространяющий горение | |
| Суффикс пожарного класса | **нг-LS**, **нг-HF** и т.д. | см. ниже |
| Количество жил × сечение | **3×2,5** = три жилы по 2,5 мм² | |
| Класс напряжения | **0,66** или **1** кВ | |

## Основные марки кабелей

| Марка | Расшифровка | Применение |
|-------|------------|-----------|
| **ВВГ** | Медь, ПВХ-изоляция, ПВХ-оболочка | Общее применение в сухих и влажных помещениях |
| **ВВГнг-LS** | То же + нераспространяющий горение, малодымный | Жилые и общественные здания, обязателен по нормам пожарной безопасности |
| **ВВГнг-HF** | То же + безгалогенный | Школы, больницы, туннели — там, где при пожаре критичны токсичные газы |
| **NYM** | Европейский аналог ВВГ: Cu, ПВХ, круглый, с наполнителем | Квартиры и офисы — удобнее в монтаже, наполнитель держит форму |
| **ПВС** | Медь, ПВХ, гибкий, многопроволочный | Удлинители, бытовые приборы — только гибкий кабель, не для стационарной проводки |
| **КВВГнг** | Контрольный кабель, медь, ПВХ, нераспространяющий | Щитовая автоматика, цепи управления |
| **ВБбШв** | Медь, ПВХ-изоляция, стальная броня, ПВХ-шланг | Прокладка в земле без дополнительной трубы |
| **КВВГ** | Контрольный, медь, ПВХ | Управляющие цепи и цепи сигнализации |

> [!СОВЕТ]
> В современном жилом строительстве выбирай **ВВГнг-LS** или **NYM** — оба соответствуют нормам. NYM чуть дороже, но его круглая форма и жёсткий наполнитель облегчают укладку в гофру.

## Категории пожарной безопасности: нг-А, нг-В, нг-С, нг-Д

По ГОСТ Р 53315-2009 кабели с пониженным распространением горения делятся на четыре категории в порядке убывания строгости требований:

| Категория | Объём кабеля в пучке при испытании | Где применяется |
|-----------|-----------------------------------|----------------|
| **нг-А** | 7 л/м (самая строгая) | Высотные здания (> 28 м), объекты с массовым пребыванием людей |
| **нг-В** | 3,5 л/м | Общественные здания, многоквартирные дома |
| **нг-С** | 1,5 л/м | Производственные и административные здания |
| **нг-Д** | 0,5 л/м | Небольшие объёмы кабелей |

> [!ВАЖНО]
> Суффикс **LS** (Low Smoke) означает низкое дымовыделение, **HF** (Halogen Free) — отсутствие галогенов при горении. Оба суффикса идут в дополнение к нг-категории: **ВВГнг(А)-LS** = категория А, малодымный. В проектах всегда указывай полную марку с категорией.

## Почему алюминиевые кабели запрещены в жилых зданиях

ПУЭ п. 7.1.34 прямо запрещает применение алюминиевых проводников сечением менее 16 мм² в жилых зданиях. Причины:

1. **Текучесть (cold flow):** алюминий под давлением контактного зажима медленно «вытекает», контакт ослабевает → нагрев → пожар.
2. **Оксидная плёнка:** быстро образуется на воздухе, имеет высокое сопротивление — переходное сопротивление растёт со временем.
3. **Гальваническая коррозия:** при контакте с медью в присутствии влаги — электрохимическая коррозия, разрушение контакта.

**Когда допустим алюминий:** ввод в здание сечением ≥ 16 мм² со специальными зажимами типа «орех» или WAGO-клеммами для алюминия/меди, промышленные установки с кабелями ≥ 16 мм².

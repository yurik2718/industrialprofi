## Что ты поймёшь после урока

Четыре типовые схемы управления освещением — от простейшего одноклавишника до перекрёстной схемы на три и более места. Разберём каждую по проводам: откуда идёт фаза, куда уходит, что переключается.

## Схема 1 — Одноклавишный выключатель

Самая простая схема. Выключатель разрывает фазный провод в цепи лампы.

```
Щит (L) → Выключатель → Лампа → Щит (N)
```

- Два провода между щитом и выключателем (или распредкоробкой): L и switched-L
- Нейтраль идёт напрямую к лампе, не через выключатель

![Схема одноклавишного выключателя](placeholder: иллюстратор рисует однолинейную схему: источник L → контакт выключателя → лампа → N обратно; рядом монтажная схема с распредкоробкой)

## Схема 2 — Двухклавишный выключатель

По сути — два независимых одноклавишника в одном корпусе. Одна фаза приходит на общий контакт, каждая клавиша управляет своей нагрузкой.

**Типичное применение:** люстра с двумя группами ламп (ближние/дальние), или два отдельных светильника в одном помещении.

```
Щит (L) → Общий контакт выключателя
                ├─ Клавиша 1 → Лампа 1 → N
                └─ Клавиша 2 → Лампа 2 → N
```

Кабель к выключателю: 3 жилы (L общий + два switched-L).

![Схема двухклавишного выключателя](placeholder: иллюстратор рисует схему: одна фаза L идёт на корпус выключателя, из него выходят два switched провода к двум лампам; N — прямой к обеим лампам)

## Схема 3 — Проходной выключатель (2-way switch)

Управление одной лампой из двух мест. Классика: коридор (у входа и у двери в комнату), лестница (внизу и вверху).

**Ключевая идея:** оба выключателя имеют по три контакта. Два контакта — «перекидные» (переключают между двумя положениями), один — «общий» (COM). Между выключателями тянется два провода — их называют **«шнурки»** или **«путевые проводники»** (travellers).

```
Щит (L) → SW1(COM) → [traveller-1 или traveller-2] → SW2(COM) → Лампа → N
```

Как это работает:
- SW1 подключает COM к одному из двух traveller-проводников
- SW2 подключает свой COM к одному из тех же двух traveller-проводников
- Лампа горит, когда оба выключателя замыкают одну и ту же «шнурку»

![Схема проходного выключателя](placeholder: иллюстратор рисует: L → COM первого выключателя; из него два провода-шнурки (красный и чёрный) уходят ко второму выключателю; из COM второго — к лампе; N прямо к лампе; под схемой — таблица состояний: оба вверху=горит, один вверху один внизу=не горит)

**Как определить проходной выключатель визуально:** у него 3 клеммы (1 COM + 2 traveller), у обычного одноклавишника — 2 клеммы.

**Маркировка проводников:** «шнурки» принято маркировать разными цветами или цветной изолентой — чтобы при обслуживании было понятно, что это промежуточные, а не нейтральные провода.

| Провод | Цвет по практике |
|---|---|
| Traveller 1 | Чёрный или красный |
| Traveller 2 | Красный или белый |
| Фаза (до SW1) | Коричневый |
| Switched фаза (от SW2) | Серый или коричневый |

> [!ВАЖНО]
> Промежуточные «шнурки» в момент переключения могут оказаться под напряжением 220 В относительно нейтрали. Маркируй их красным — это не N и не PE, не подключай к ним ничего лишнего.

## Схема 4 — Перекрёстный выключатель (3 и более мест)

Для трёх и более точек управления. Схема: два проходных выключателя (на концах) + один или несколько перекрёстных (кресто-образных, 4 контакта) в середине.

```
SW1(проходной) → SW_middle(перекрёстный) → SW2(проходной) → Лампа
```

Перекрёстный выключатель имеет 4 клеммы. Он меняет местами два провода (travellers) — это и есть «перекрещивание». Сколько дополнительных мест управления нужно — столько перекрёстных выключателей добавляется в середину цепи.

![Схема с перекрёстным выключателем](placeholder: иллюстратор рисует цепочку: L → проходной SW1 → перекрёстный SW_mid → проходной SW2 → лампа; между SW1 и SW_mid два провода, между SW_mid и SW2 тоже два провода; под схемой — таблица из 8 строк с состояниями трёх переключателей)

## Сводная таблица

| Схема | Кол-во выключателей | Провода между коробкой и SW | Применение |
|---|---|---|---|
| Одноклавишный | 1 (2 клеммы) | 2 жилы | Одна комната, одна лампа |
| Двухклавишный | 1 (3 клеммы + COM) | 3 жилы | Две нагрузки, одно место |
| Проходной | 2 (по 3 клеммы) | 3 жилы к SW1, 3 к SW2 | Два места управления |
| С перекрёстным | 2 проходных + N перекрёстных | 3 + 3 + (N×2) | Три и более мест |

## Типовая ошибка

Самая частая ошибка при монтаже проходного выключателя — подключить «шнурки» к клеммам COM вместо traveller-клемм. Результат: свет либо не включается вообще, либо горит в одном положении из четырёх. Проверь схему по документации на конкретную модель выключателя — обозначения клемм у разных производителей отличаются (L1/L2, 1/2, COM/L).

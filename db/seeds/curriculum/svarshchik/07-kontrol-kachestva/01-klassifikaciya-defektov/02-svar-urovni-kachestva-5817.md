ГОСТ Р ИСО 5817-2021 устанавливает конкретные цифры: какого размера дефект ещё допустим, а при каком шов уходит на переварку — это и есть решение «годен / не годен».
---
## Зачем нужны уровни качества

Любой шов содержит микронесовершенства — вопрос в том, насколько они велики и насколько критична конструкция. ГОСТ Р ИСО 5817 формализует этот вопрос через три уровня качества:

- **Уровень B (высший)** — минимальные допустимые отклонения; применяется там, где дефект может вызвать катастрофу
- **Уровень C (средний)** — стандарт для большинства несущих конструкций
- **Уровень D (базовый)** — наименее жёсткие требования; для ненагруженных или легко заменяемых деталей

Уровень назначается конструктором в проекте или заказчиком в техническом задании — не сварщиком и не мастером. Твоя задача: знать, какой уровень указан в ПТД (WPS), и соблюдать его.

> [!ВАЖНО]
> ГОСТ Р ИСО 5817-2021 распространяется на сталь, никель, титан и их сплавы. Для алюминиевых сплавов — отдельный стандарт ИСО 10042.

## Типичные области применения каждого уровня

| Уровень | Типичные объекты |
|---------|-----------------|
| B — высший | Сосуды давления, трубопроводы нефтегаза (НГК), несущие мостовые конструкции, атомные объекты |
| C — средний | Строительные металлоконструкции, промышленное оборудование, крановые балки, вагоны |
| D — базовый | Заборы, нагрузочные поддоны, прицепы малой грузоподъёмности, ненагруженные кронштейны |

## Как читать таблицу стандарта

ГОСТ Р ИСО 5817 содержит таблицу со всеми дефектами по кодам ИСО 6520-1. Для каждого дефекта и каждого уровня (B/C/D) указано:
- допустимый размер (высота, длина, площадь) — часто в виде формулы через толщину основного металла *t*
- или категория «не допускается»

Обозначения в таблице:
- **h** — высота (глубина) дефекта
- **l** — длина дефекта
- **t** — толщина свариваемого металла
- **d** — диаметр поры
- **Не допускается** — любой размер дефекта является браком

## Критерии для ключевых дефектов

### Подрез (код 5011)

Подрез — канавка вдоль кромки шва — один из самых частых поверхностных дефектов.

| Уровень | Допустимая глубина |
|---------|--------------------|
| D | h ≤ 0,2 мм при t ≤ 3 мм; h ≤ 0,5 мм при t > 3 мм |
| C | h ≤ 0,1 мм при t ≤ 3 мм; h ≤ 0,2 мм при t > 3 мм |
| B | Не допускается (практически любой подрез отправляет на ремонт) |

На практике для уровня B подрез проверяется глубиномером: даже 0,1 мм — граничное значение.

### Поры — единичная (код 2011) и скопление (код 2013)

Оценивается по диаметру одиночной поры и суммарной площади пор на единицу длины шва.

| Параметр | Уровень D | Уровень C | Уровень B |
|----------|-----------|-----------|-----------|
| Максимальный диаметр одиночной поры | 0,4t, но не более 5 мм | 0,3t, но не более 4 мм | 0,2t, но не более 3 мм |
| Суммарная площадь пор на 100 мм длины | ≤ 2% | ≤ 1% | Не допускается скопление |

### Несоосность (смещение кромок, код 507)

| Уровень | Допустимое смещение |
|---------|---------------------|
| D | ≤ 0,25t, но не более 5 мм |
| C | ≤ 0,15t, но не более 4 мм |
| B | ≤ 0,1t, но не более 3 мм |

### Превышение усиления шва (избыточная выпуклость, код 502)

| Уровень | Допустимая высота усиления |
|---------|-----------------------------|
| D | ≤ 1 мм + 0,25b (где b — ширина шва) |
| C | ≤ 1 мм + 0,15b |
| B | ≤ 1 мм + 0,1b |

### Непровар корня (код 4013)

| Уровень | Допуск |
|---------|--------|
| D | Допускается при h ≤ 0,2t, но не более 2 мм, и длине не более 25 мм на 100 мм шва |
| C | Не допускается |
| B | Не допускается |

> [!ПРИМЕР]
> Шов стыкового соединения балки, t = 12 мм, уровень C. Измеренная глубина подреза — 0,25 мм. Допуск по уровню C: h ≤ 0,2 мм. **Вывод: не годен.** Требуется зашлифовать подрез до плавного перехода (если это не выводит сечение за пределы минимальной толщины) или переварить участок.

## Процедура приёмки: «годен» или «не годен»

```
1. ВИК → зафиксировать все видимые дефекты
2. Для каждого дефекта → определить код по ИСО 6520-1
3. Найти в таблице ИСО 5817 → применить критерий для заданного уровня (B/C/D)
4. Если дефект в пределах допуска → «Годен» (годен по ВИК)
5. Если дефект вне допуска → «Не годен» → оформить заключение → ремонт
6. После ремонта → повторный контроль того же участка
```

Помимо ВИК, инспектор назначает методы НК (УЗК, РК, ПВК, МПК) по требованию проекта или нормативного документа. Критерии приёмки по НК — в отдельных стандартах, но ссылаются на те же коды ИСО 6520-1.

## Ошибки при работе со стандартом

**Ошибка 1: Использовать допуск не своего уровня.** Уровень указан в чертеже или WPS — только он действует. Нельзя применить более мягкий уровень D, если проект требует C.

**Ошибка 2: Не учитывать «t»** при расчёте допуска. Один и тот же дефект может быть допустим для листа 20 мм и недопустим для листа 3 мм.

**Ошибка 3: Принимать решение «годен» только по внешнему виду.** Стандарт устанавливает критерии ВИК, но внутренние дефекты (поры, непровар, несплавление) выявляются только НК.

> [!СОВЕТ]
> Перед сваркой ответственного стыка прочитай WPS: найди строку «уровень качества» или «acceptance criteria». Это определяет, насколько «идеально» должен выглядеть твой шов. Разница между уровнем C и B — это разница между «всё нормально» и «бракуем».

> [!ПРОВЕРЬ]
> 1. Какой уровень качества назначают для сосудов давления и трубопроводов нефтегаза?
> 2. Допускается ли непровар корня шва на уровне C по ИСО 5817?
> 3. Как расшифровывается обозначение «h ≤ 0,2t, не более 2 мм» применительно к дефекту подреза на листе толщиной 10 мм?
> 4. Кто назначает уровень качества — сварщик, мастер или конструктор?

## Задание

Возьми один из своих сваренных стыковых образцов (или воспользуйся описанием дефектов с предыдущего урока). Условие: уровень качества C, толщина металла 8 мм. Для каждого дефекта, который ты обнаружил при ВИК, запиши: (1) код по ИСО 6520-1, (2) измеренный размер, (3) допустимый размер по уровню C из таблицы ИСО 5817, (4) решение «Годен» / «Не годен». Если нет образца — выполни то же задание для следующих воображаемых дефектов: подрез глубиной 0,3 мм, одиночная пора диаметром 2 мм, смещение кромок 1,5 мм.

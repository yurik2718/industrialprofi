## О чём эта практика

Это настольное упражнение: ты разбираешь сварочные обозначения, составляешь список параметров и заполняешь документы без сварочного оборудования. Такой разбор — обязательная часть квалификационного экзамена (ЕТКС 4-й разряд и выше требует умения читать чертёж).

## Задание 1: Расшифровка обозначений на чертеже

Перед тобой четыре сварочных обозначения с чертежа сварной рамы. Для каждого запиши: тип соединения, способ сварки, размер шва, вспомогательные символы и их значение.

**Обозначение А:**
```
k6 Т3 ГОСТ 5264-80
```

**Обозначение Б:**
```
С8 ГОСТ 5264-80
```

**Обозначение В:**
```
k8 Т6 ГОСТ 14771-76 ○
```

**Обозначение Г:**
```
k5 Н1 ГОСТ 5264-80 ▷
```

Заполни таблицу:

| Обозначение | Тип соединения | Способ сварки | Размер (катет или тип разделки) | Вспомогательный символ и его смысл |
|------------|---------------|--------------|--------------------------------|-------------------------------------|
| А | ? | ? | ? | (нет) |
| Б | ? | ? | ? | (нет) |
| В | ? | ? | ? | ? |
| Г | ? | ? | ? | ? |

> [!СОВЕТ]
> Для расшифровки Обозначения Б нужно открыть ГОСТ 5264-80, найти соединение С8 и прочитать: толщины металла, форму разделки, наличие зазора. Это реальная работа с нормативным документом.

## Задание 2: Выбор режима сварки по чертежу

По чертежу задано:
- Соединение: С8 по ГОСТ 5264-80 (стыковое, двусторонний шов, без скоса кромок, для толщин 4–8 мм)
- Основной металл: сталь Ст3сп, толщина листа 8 мм
- Способ сварки: РД (ручная дуговая)
- Уровень качества: C (ГОСТ Р ИСО 5817)

Ответь на вопросы:

**2.1.** Какой диаметр электрода выбрать для корневого прохода? Для заполняющего? (Подсказка: для РД диаметр электрода ≈ толщина металла, но для корня всегда меньше.)

**2.2.** Электрод УОНИ 13/45, Ø3,2 мм. Из паспорта электрода: рекомендуемый ток 80–110 А, постоянный ток обратной полярности. Запиши настройки сварочного аппарата для корневого прохода.

**2.3.** Нужен ли предварительный подогрев для стали Ст3сп толщиной 8 мм при РД? (Подсказка: углеродный эквивалент Ст3сп — около 0,25–0,30; подогрев требуется при Сэкв > 0,45.)

**2.4.** После завершения шва какие параметры ты проверишь при ВИК? Запиши список не менее 4 параметров и допуски по уровню C и толщине 8 мм.

## Задание 3: Список документов для ответственного шва

Условие: ты сварил стыковой шов трубы Ду 108, t = 6 мм, сталь 09Г2С, способ РД, на объекте, подконтрольном Ростехнадзору (категория трубопровода II).

Составь полный перечень документов, которые должны быть оформлены по этому шву. Для каждого документа укажи: название, кто составляет, кто подписывает.

Используй следующий шаблон:

| № | Документ | Кто составляет | Кто подписывает |
|---|---------|---------------|----------------|
| 1 | | | |
| 2 | | | |
| 3 | | | |
| ... | | | |

> [!ВАЖНО]
> Минимально ожидаемый список — не менее 5 позиций. Подсказка: подумай о сертификатах на материалы, журнале, паспорте соединения, актах контроля.

## Задание 4: Выявление нарушения по журналу

В журнале сварочных работ обнаружена следующая запись:

| Дата | Шов № | Способ | Электрод | Ток, А | Клеймо | ВИК |
|------|--------|-------|---------|--------|--------|-----|
| 12.09.2024 | СШ-15 | РД | УОНИ 13/55 Ø3,2 | 160 | СВ23 | Годен |

ПТД для этого соединения указывает: «Способ РД, электрод УОНИ 13/55 Ø3,2 мм, ток корневого прохода 80–100 А, ток заполняющих проходов 100–130 А».

Найди нарушения в записи и ответь:
1. Какое нарушение допущено?
2. Нужно ли переваривать шов? Почему?
3. Что мастер должен был сделать при обнаружении записи?

## Эталонные ответы

### К заданию 1

| Обозначение | Тип соединения | Способ сварки | Размер | Вспомогательный символ |
|------------|---------------|--------------|--------|----------------------|
| А | Тавровое (Т3) | РД (ГОСТ 5264-80) | Катет 6 мм | — |
| Б | Стыковое (С8) | РД (ГОСТ 5264-80) | По ГОСТ (без скоса, 4–8 мм) | — |
| В | Тавровое (Т6) | МИГ/МАГ (ГОСТ 14771-76) | Катет 8 мм | ○ — шов по периметру |
| Г | Нахлёсточное (Н1) | РД (ГОСТ 5264-80) | Катет 5 мм | ▷ — монтажный шов |

### К заданию 2

**2.1.** Корень: Ø3,2 мм; заполнение: Ø4,0 мм (для металла 8 мм).

**2.2.** Ток 90 А (в диапазоне 80–110 А), постоянный ток, полярность обратная (плюс на электроде).

**2.3.** Подогрев не требуется: Ст3сп Сэкв ≈ 0,27, что ниже порога 0,45.

**2.4.** ВИК параметры (уровень C, t = 8 мм): высота усиления ≤ 2,2 мм; ширина шва (по чертежу ±2 мм); глубина подреза ≤ 0,16 мм; смещение кромок ≤ 1,2 мм (0,15×8).

### К заданию 4

**Нарушение:** ток 160 А превышает максимум по ПТД (130 А для заполняющих, 100 А для корня). Вероятно, корень варили на токе заполнения — нарушение ПТД.

**Переварка:** решение принимает технолог совместно с ответственным за сварку ИТР. Если НК не показал дефектов (ВИК «Годен», и если будет назначен УЗК — тоже «Годен»), то шов может быть принят по фактическим механическим свойствам — но это требует документального решения (отклонение). На практике на строгих объектах НГК такое нарушение ведёт к вырезке.

**Мастер:** обязан был: (а) зафиксировать нарушение, (б) потребовать объяснение от сварщика, (в) инициировать назначение дополнительного НК (УЗК), (г) сделать запись в журнале.

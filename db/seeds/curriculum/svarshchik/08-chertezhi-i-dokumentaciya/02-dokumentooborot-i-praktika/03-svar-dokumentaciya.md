## Зачем нужна документация

Сварной шов — элемент, скрытый после монтажа или покраски. Единственный способ доказать, что он выполнен правильно — документы. На объектах Ростехнадзора (трубопроводы, сосуды, резервуары, мосты) отсутствие документации на сварку — основание для запрета ввода в эксплуатацию.

Документы сварочного производства выполняют три функции:
1. **Прослеживаемость** — кто, что, когда, чем и по какой технологии варил
2. **Доказательность** — результаты контроля, подтверждающие качество
3. **Ответственность** — подпись под документом означает личную ответственность

## Журнал сварочных работ

**Журнал сварочных работ** — основной производственный документ, который ведётся на каждом объекте. Форму обычно устанавливает организация, но минимальный набор граф стандартен:

| Графа | Содержание |
|-------|-----------|
| Дата | Дата выполнения шва |
| Номер шва | Согласно исполнительной схеме (ИС) |
| Наименование соединения | Труба Ø108×6 мм / Лист 12 мм |
| Основной металл | Марка стали, плавка, сертификат |
| Присадочный материал | Марка, партия, номер сертификата |
| Номер ПТД/WPS | Ссылка на применённую ПТД |
| Клеймо сварщика | Личное клеймо (буква+цифра, аналог подписи) |
| Позиция сварки | Н1, Н2, Н45° и т.д. |
| Режим сварки | Ток, напряжение (или ссылка на ПТД) |
| Температура подогрева | Фактическая, измеренная |
| Результат ВИК | Годен / Не годен, дата, подпись |
| Результат НК | Метод, протокол №, дата, решение |
| Ремонт | Дата, метод, повторный НК |

Журнал хранится у мастера или ответственного за сварку ИТР (инженерно-технического работника). Сварщик **подписывает** каждую запись своим клеймом.

> [!ВАЖНО]
> Личное **клеймо сварщика** — не печать, а уникальный буквенно-цифровой идентификатор (например, «СВ42»), присваиваемый предприятием. Клеймо ставится на шов или в документ вместо подписи. По клейму можно установить конкретного исполнителя через много лет после сдачи объекта.

## Паспорт сварного соединения

На ответственных соединениях (сосуды давления, трубопроводы категорий I–III, мостовые конструкции) для каждого шва оформляется **паспорт сварного соединения** — отдельный документ, включающий:

- Идентификатор соединения (номер по ИС)
- Чертёж или схему стыка
- Данные о сварщике: ФИО, клеймо, номер удостоверения НАКС
- Применённые материалы: металл (марка, плавка) + присадочный (марка, партия)
- Применённая ПТД (номер, дата)
- Фактические режимы (если требуется по проекту)
- Результаты всех видов контроля: ВИК + ПВК/МПК + УЗК/РК — акты и заключения
- Ремонты (при наличии): дата, объём, повторный НК
- Итоговое заключение: «Соединение допущено к эксплуатации»
- Подписи: сварщик, мастер (ответственный ИТР), специалист НК

Паспорта сварных соединений подшиваются в общий паспорт объекта (например, паспорт трубопровода или резервуара).

## Исполнительная схема (ИС)

Исполнительная схема — чертёж или схема, на которой показаны все сварные соединения объекта с нумерацией. Каждому шву присваивается уникальный номер (например, «СШ-23» или «С-047»).

ИС разрабатывает инженер-проектировщик или технолог. По этой нумерации шов идентифицируется в журнале и паспортах.

## Сертификаты материалов

Перед использованием сварочных материалов (электродов, проволоки, газа) и основного металла на ответственных работах обязательно проверяют сертификат качества (паспорт изделия):

| Документ | Что содержит |
|----------|-------------|
| Сертификат на трубу/лист | Марка стали, плавка, хим. состав, мех. свойства, ГОСТ |
| Паспорт на электроды | Марка, ГОСТ, завод, партия, дата изготовления, хим. состав покрытия |
| Паспорт на сварочную проволоку | Марка, ГОСТ, плавка, хим. состав |
| Паспорт на защитный газ | Марка смеси, объём, анализ (% CO₂, O₂, примесей) |

Номера сертификатов вносятся в журнал сварочных работ — так создаётся цепочка прослеживаемости «металл → электрод → шов → контроль».

## Удостоверение сварщика НАКС

Удостоверение сварщика НАКС — документ, подтверждающий, что данный сварщик аттестован на выполнение конкретного вида работ.

### Что указано в удостоверении

| Поле | Пример |
|------|--------|
| ФИО | Иванов Иван Иванович |
| Дата рождения | 15.05.1990 |
| Номер удостоверения | НАКС-РД-М01М02-Н1Н2-2022-00342 |
| Способы сварки | РД (111) |
| Группы материалов | М01, М02 |
| Типы соединений | Стыковые (С), Тавровые (Т) |
| Позиции сварки | Н1, Н2, Н45° |
| Группы объектов | НГК, ОПО |
| Срок действия | До 15.06.2026 |
| Дата выдачи и АЦ | 15.06.2024, ООО «СвармасТест», г. Казань |

### Срок действия и продление

- Стандартный срок: **2 года** от даты аттестации
- Продление: за 1–3 месяца до истечения срока — повторное практическое испытание в аттестационном центре
- Досрочное прекращение: при перерыве в работе по данному виду сварки более 6 месяцев

### Как проверить подлинность

На сайте naks.ru есть реестр аттестованных сварщиков. Работодатель и заказчик имеют право проверить удостоверение через реестр. Подделка удостоверения — уголовная ответственность.

## Акт визуального и измерительного контроля (ВИК)

Акт ВИК оформляется после финального осмотра шва. Содержит:
- Идентификатор соединения
- Применявшийся нормативный документ (ИСО 5817, уровень)
- Инструменты контроля (шаблон УШС-3, лупа, глубиномер)
- Перечень проверенных параметров с измеренными значениями
- Обнаруженные дефекты (код по ИСО 6520-1, размер)
- Решение: Годен / Не годен
- Подпись специалиста ВИК, дата

## Протокол (заключение) неразрушающего контроля

После УЗК или РК специалист НК оформляет протокол (заключение), который содержит:
- Метод и объём контроля
- Применявшийся стандарт
- Оборудование и эталоны (дефектоскоп, тип датчика, IQI/КЧ)
- Обнаруженные дефекты (код, размер, глубина, расположение)
- Решение: Годен / Не годен
- Подпись специалиста НК (с указанием уровня аттестации), дата

Все акты и протоколы подшиваются к паспорту сварного соединения.

## Документооборот: кто что создаёт

| Документ | Кто создаёт | Кто подписывает |
|---------|------------|----------------|
| Журнал сварочных работ | Мастер / ответственный ИТР | Сварщик (клеймо), мастер |
| Паспорт сварного соединения | Мастер / технолог | Сварщик, мастер, НК-специалист |
| Исполнительная схема | Инженер ПТО | ИТР, приёмщик |
| Акт ВИК | Специалист ВИК | Специалист ВИК |
| Протокол УЗК/РК | Специалист НК II уровня | Специалист НК |
| Журнал прокалки электродов | Мастер / сварщик | Мастер |

> [!ПРИМЕР]
> Схема документооборота для одного ответственного трубного стыка:
> 1. Сварщик получает ПТД + сертификаты на трубу и электроды — проверяет, расписывается в ознакомлении.
> 2. Мастер фиксирует в журнале: номер шва, материалы, клеймо сварщика, дату.
> 3. После сварки — акт ВИК (специалист ВИК или мастер с полномочиями).
> 4. Если ВИК «Годен» — заявка на УЗК/РК; специалист НК выдаёт заключение.
> 5. Всё подшивается в паспорт стыка → в паспорт объекта.

> [!ПРОВЕРЬ]
> 1. Что такое клеймо сварщика и для чего оно используется?
> 2. Какой документ содержит полную информацию об одном конкретном сварном соединении?
> 3. Срок действия удостоверения НАКС — сколько лет?
> 4. Что произойдёт с действием удостоверения НАКС при перерыве в работе по данному виду сварки более 6 месяцев?

## Задание

Составь макет паспорта сварного соединения для следующей ситуации: ты сварил стыковой шов трубы Ду 108, t = 6 мм, сталь 09Г2С, способ РД, электрод УОНИ 13/55 Ø3,2 мм. ВИК — годен, УЗК — годен. Заполни все основные поля (можно вымышленными данными: номер шва, дата, ФИО, клеймо «СВ17», номер ПТД). Обозначь, каких документов (актов, протоколов) не хватает для полного паспорта.

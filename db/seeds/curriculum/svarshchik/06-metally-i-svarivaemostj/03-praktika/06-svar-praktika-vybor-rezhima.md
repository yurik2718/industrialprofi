Карта выбора материала и режима — навык, без которого нельзя работать самостоятельно: здесь три реальные задачи на анализ металла, присадки, газа и подогрева без доступа к оборудованию.
---
## Задание

Это настольное упражнение (бумага и карандаш): никакого оборудования не нужно. Реши три производственные задачи, используя знания из курсов по материалам и технологиям. Запиши решения в рабочий журнал — с обоснованием каждого выбора.

---

### Сценарий 1: Сварка балки в мороз

**Условие:**  
Тебе дают задание сварить **балку из стали 09Г2С, толщина полок 30 мм**. Работа ведётся на улице, температура воздуха **–10°C**. Метод сварки — ручная дуговая (РД).

**Известный химический состав 09Г2С (типичный):**  
C = 0,09%; Mn = 1,70%; Si = 0,60%; Cr = 0,30%; Ni = 0,30%; Cu = 0,30%; Mo = 0.

**Задача — ответить на следующие вопросы:**

1. **Рассчитай Сэкв** для данной стали по формуле:  
   Сэкв = C + Mn/6 + (Cr+Mo+V)/5 + (Ni+Cu)/15

2. **К какой группе свариваемости** относится сталь по рассчитанному Сэкв?

3. **Нужен ли предварительный подогрев?** Учти: толщина 30 мм и температура –10°C (два отягощающих фактора).

4. **До какой температуры нагреть металл** (используй таблицу из урока по подогреву или формулу Сефериана)?

5. **Какой электрод выбрать?** Выбери марку и тип покрытия, обоснуй. Нужна ли прокалка?

6. **Какие меры принять** при вынужденном перерыве на 30 минут (смена электрода кончилась, нужно съездить за материалом)?

---

**Ответы: что должно получиться**

> Ниже — эталонный ответ. Реши сначала сам, потом сверяй.

<details>
<summary>▶ Эталонный ответ сценария 1 (нажми для просмотра)</summary>

1. **Сэкв** = 0,09 + 1,70/6 + (0,30+0+0)/5 + (0,30+0,30)/15  
   = 0,09 + 0,283 + 0,060 + 0,040 = **0,473**

2. **Свариваемость:** Сэкв 0,45–0,55 → **ограниченная** (по стандартной классификации).  
   Примечание: 09Г2С с нормальным химсоставом обычно даёт Сэкв ~0,37–0,42. При 0,47 (высокий конец) — ближе к «ограниченной». Практика: ориентироваться на реальный сертификат.

3. **Подогрев нужен:** толщина 30 мм + мороз –10°C — оба фактора требуют подогрева. При Сэкв > 0,40, толщине > 20 мм и температуре < +5°C — подогрев обязателен.

4. **Температура подогрева:** T = 350 × √(0,473 – 0,25) = 350 × √0,223 = 350 × 0,472 = **165°C**.  
   Округляем до 150–200°C. При –10°C на улице — верхняя граница, т.е. **200°C**.

5. **Электрод:** основной тип (Б) — УОНИ 13/55 или УОНИ 13/45 Ø3–4 мм. Основные электроды дают низкое содержание водорода — это ключевое при работе с ограниченно свариваемыми сталями. **Прокалка:** 350–400°C, 1–2 часа перед работой (обязательно!).

6. **При перерыве:** накрыть шов теплоизолирующим материалом (стекловата, асбестовый картон). Если перерыв > 20–30 мин — проверить температуру; при остывании ниже 150°C — снова подогреть до 200°C, только потом продолжать.

</details>

### Сценарий 2: Нержавеющий лист, внахлёст

**Условие:**  
Тебе дают задание: соединить нахлёсточным швом два листа **нержавеющей стали 08Х18Н10 (аналог AISI 304), толщина 3 мм**, для корпуса агрегата пищевого предприятия. Метод — МИГ/МАГ.

**Задача — ответить:**

1. **Какой газ использовать?** Можно ли CO₂? Почему?

2. **Какую присадочную проволоку** выбрать? Почему нельзя использовать Св-08Г2С?

3. **Какое максимальное напряжение** на полуавтомате при толщине 3 мм и проволоке Ø0,8 мм? Режим переноса?

4. **Какая максимальная межпроходная температура** допустима для нержавейки?

5. **Как подготовить поверхность** перед сваркой (два обязательных шага)?

6. **Чем опасна сенсибилизация** для данного изделия (пищевое предприятие)?

---

**Ответы:**

<details>
<summary>▶ Эталонный ответ сценария 2 (нажми для просмотра)</summary>

1. **Газ:** Ar+2%CO₂ или чистый аргон. **CO₂ нельзя:** высокое содержание активного газа увеличивает тепловложение и насыщение шва углеродом → риск сенсибилизации и ухудшение коррозионной стойкости.

2. **Проволока:** Св-04Х19Н9 (для 304/08Х18Н10) или Св-04Х19Н11 (немного более легированная). **Св-08Г2С нельзя:** состав принципиально другой (высокий C, нет Cr и Ni) — шов будет без коррозионной стойкости, возможны горячие трещины из-за несовместимости кристаллизации.

3. **Напряжение:** 16–19 В (режим КЗ — короткое замыкание). 3 мм + Ø0,8 мм → только КЗ. Струйный перенос недопустим при малой толщине — прожог.

4. **Межпроходная температура:** не более **150°C**. Выше — риск сенсибилизации.

5. **Подготовка:** (а) обезжиривание ацетоном или спиртом; (б) зачистка **нержавеющей** щёткой (не стальной — частицы железа дадут ржавчину на нержавейке). Оба шага — непосредственно перед сваркой.

6. **Сенсибилизация на пищевом предприятии:** потеря коррозионной стойкости в зоне шва → при контакте с кислыми продуктами (кетчуп, молочные кислоты, рассол) — межкристаллитная коррозия → ионы тяжёлых металлов в продукте. Санитарная угроза + нарушение норм пищевого производства.

</details>

### Сценарий 3: Алюминий АМг5, стыковой шов

**Условие:**  
Тебе нужно соединить **два листа алюминиевого сплава АМг5 (5083), толщина 5 мм**, стыковым швом С2 по ГОСТ 14771-76. Доступен полуавтомат МИГ с аргоном.

**Задача — ответить:**

1. **К какой серии** относится сплав АМг5? Хорошо ли он сваривается?

2. **Какую присадочную проволоку** выбрать?

3. **Какой газ и диаметр проволоки?**

4. **Какой лайнер** должен быть в горелке МИГ и почему?

5. **Перечисли три шага подготовки поверхности** перед сваркой.

6. **Можно ли нагреть деталь газовой горелкой «для лучшего прогрева»?** Почему да или нет?

7. **Как определить**, что дуга работает правильно (какой визуальный признак катодного очищения при МИГ DC+ на алюминии)?

---

**Ответы:**

<details>
<summary>▶ Эталонный ответ сценария 3 (нажми для просмотра)</summary>

1. **Серия 5xxx** (Al-Mg сплавы). Свариваемость **отличная** — АМг5 относится к лучшим конструкционным алюминиевым сплавам для сварки.

2. **Присадка:** Св-АМг5 (или по ГОСТ Р ИСО 18273 — ER5356). Совпадение по составу обеспечивает хорошую коррозионную стойкость шва.

3. **Газ:** чистый аргон (Ar, марка «высший»). **Диаметр проволоки:** Ø1,0 мм для металла 5 мм (Ø0,8 мм слишком мягкая для стабильной подачи).

4. **Лайнер:** тефлоновый (пластиковый или PTFE). Алюминиевая проволока мягкая — в стальном лайнере она смнётся и застрянет. Также рекомендуется укороченный шланговый пакет (не более 3 м).

5. **Три шага подготовки:**  
   (а) Обезжиривание ацетоном или изопропиловым спиртом.  
   (б) Механическая зачистка **нержавеющей щёткой** (отдельной, только для алюминия — не использовать после стали).  
   (в) Прогреть деталь до температуры помещения (не допускать конденсата при внесении холодного металла в тёплое помещение). Сварку начать сразу — не оставлять зачищенную поверхность дольше 30 минут.

6. **Небольшой предварительный нагрев допустим** для толстого алюминия (> 8–10 мм): до 80–120°C газовой горелкой равномерно. Для 5 мм — **не нужен**, но не вреден, если не перегреть. Перегрев (> 120–150°C) → оксид наросёт снова, деталь задымится, а у 5xxx сплавов при перегреве возможно локальное оплавление по границам зёрен.

7. **Визуальный признак:** вокруг сварного шва видна **светлая матовая зона** — зачищенный алюминий без оксидной плёнки шириной 5–10 мм. Если металл вокруг шва остаётся тёмно-серым или чёрным — катодного очищения нет, шов с включениями оксида.

</details>

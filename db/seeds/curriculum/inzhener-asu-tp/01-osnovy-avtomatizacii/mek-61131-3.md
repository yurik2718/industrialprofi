МЭК 61131-3 — **единый стандарт** для всех ПЛК. Знание стандарта позволяет переходить между платформами (Siemens, ОВЕН, Schneider) без переучивания.

---

## Пять языков программирования ПЛК

1. **LD** (Ladder Diagram) — релейные диаграммы, самый популярный
2. **FBD** (Function Block Diagram) — функциональные блоки
3. **ST** (Structured Text) — текстовый, похож на Pascal
4. **IL** (Instruction List) — ассемблер ПЛК, устаревает
5. **SFC** (Sequential Function Chart) — последовательностные диаграммы

## Организация программы

Стандарт определяет **POU** (Program Organization Unit):
- **Program** — основная программа
- **Function** — не хранит состояние (чистая функция)
- **Function Block** — хранит состояние между вызовами (экземпляры)

## Типы данных

Стандартные: `BOOL`, `INT`, `REAL`, `TIME`, `STRING`

> На практике 90% промышленных программ написаны на **LD** и **ST**. FBD используется реже, IL устарел, SFC — для рецептурных процессов.

## Задание

Напишите одну и ту же логику на **трёх языках** (LD, FBD, ST):

> Управление насосом: пуск при уровне в баке **ниже 30%** И нет аварии. Останов при уровне **выше 80%** ИЛИ аварийный останов.

Сравните читаемость каждого варианта.

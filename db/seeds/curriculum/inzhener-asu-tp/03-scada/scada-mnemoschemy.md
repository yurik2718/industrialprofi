Плохая мнемосхема — это **пропущенная авария**. Оператор должен за 3 секунды понять состояние процесса.

---

## Принципы проектирования (ISA-101)

- **Серый фон** — не чёрный! Снижает утомляемость оператора
- **Цвет только для отклонений:**
  - Красный — авария
  - Жёлтый — предупреждение
  - Зелёный — используется минимально (норма = серый)
- **Динамические элементы** — уровни в баках, положения задвижек, тренды в реальном времени

## Навигация

Иерархия от обзорного экрана к детальным:

1. **Обзорный экран** — весь процесс, ключевые KPI
2. **Зоны** — отдельные участки (насосная, фильтры, реагенты)
3. **Детальные** — конкретное оборудование с параметрами

> **Правило 3 кликов:** оператор должен добраться до любого параметра максимум за 3 клика.

## Задание

Спроектируйте **мнемосхему для насосной станции**:
- 2 насоса (рабочий + резервный)
- Бак-накопитель с датчиком уровня
- Задвижка на выходе

Нарисуйте макет экрана с учётом принципов ISA-101. Укажите динамические элементы и цвета для состояний.

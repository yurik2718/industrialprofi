## Три группы — одна логика

Профессиональный набор сантехника делится на три функциональные группы. Если помнишь эту структуру, никогда не забудешь нужный инструмент на объекте:

1. **Резать и гнуть** — подготовить трубу нужной длины и формы
2. **Соединять** — выполнить соединение (резьба, сварка, пресс)
3. **Измерять и проверять** — убедиться, что всё сделано правильно

## Группа 1: резать и гнуть

### Труборез роликовый

Основной инструмент для резки стальных, медных и PPR-труб. Работает по принципу кольцевого реза: роликовый нож обкатывается вокруг трубы с постепенным затягиванием. Рез — точно перпендикулярный, без заусенцев снаружи (но образуется **внутренняя заусенина** — её снимает конусная развёртка на корпусе резака).

Типоразмеры трубореза — по диаметру труб: для ½"–2" (обычный) и для малых диаметров ¼"–1" (для меди). Есть варианты для нержавейки.

> [!СОВЕТ]
> При резке закручивай ролик не «до упора», а **постепенно — по 1/4 оборота на каждый круг**. Сильное затягивание сразу деформирует трубу и быстро тупит ролик.

### Ножницы для полимерных труб

Для PPR, металлопластика и ПНД диаметром до 63 мм. Роторные ножницы дают чистый перпендикулярный рез за один нажим. Обычные строительные ножницы или болгарка — недопустимая замена: косой рез не сварится качественно.

Для труб DN50 и выше — роторный резак с трещоткой (требует нескольких нажимов).

### Трубогиб

Для медных и металлопластиковых труб — позволяет изогнуть трубу без заломов и сужения сечения.

- **Пружинный трубогиб** — надевается на трубу снаружи (или вставляется внутрь), гнёшь руками. Прост, дёшев, подходит для мягкой меди и металлопластика DN15–DN20. Минус: сложно получить точный угол.
- **Рычажный трубогиб** — специальный шаблон с рычагом фиксирует угол. Точный угол от 15° до 180°. Для меди и металлопластика DN15–DN25.

> [!ВАЖНО]
> Минимальный радиус гиба металлопластиковой трубы — 5 диаметров трубы. DN16 гнуть не тоньше R = 80 мм. Нарушение — залом и перекрытие сечения.

### Болгарка (УШМ)

Используется для резки стальных труб большого диаметра, когда труборез неприменим. **Только в крайнем случае** для бытовых диаметров: рез под углом, много заусенцев, шум и искры. Для PPR и металлопластика — никогда.

## Группа 2: соединять

### Плашка + плашкодержатель (вороток)

Для нарезки наружной резьбы на стальных трубах. Плашка — кольцо с режущими зубьями, плашкодержатель — рукоятка для вращения.

Самые ходовые размеры резьбы для сантехники: **G½" (Dy15), G¾" (Dy20), G1" (Dy25)** — по ГОСТ 6111 и ГОСТ 6357 (трубная резьба Whitworth).

> [!ВАЖНО]
> Трубная резьба G обозначается в дюймах и **не совпадает** с метрической резьбой M. Никогда не пытайся накрутить метрическую гайку на трубную резьбу — испортишь и гайку, и трубу.

Техника нарезки: плашку смазывают машинным маслом, каждые 1–1.5 оборота вперёд делают полоборота назад для ломки стружки. Глубина нарезки: 5–6 ниток для ½", 6–7 ниток для ¾".

### Электрический клупп

Профессиональный инструмент для нарезки резьбы на объёмах. Работает как дрель: зажимаешь трубу в тисках, вставляешь головку с плашкой, нажимаешь кнопку — резьба нарезается автоматически. В 5–8 раз быстрее ручной плашки, резьба ровнее. Для постоянной работы — незаменим.

### Паяльник для PPR (сварочный аппарат)

Нагревательный прибор с плоскими насадками под конкретные диаметры (20, 25, 32 мм и т.д.). Рабочая температура: **260°С** — это стандарт для PP-R.

Комплект для сварки PPR:
- Паяльник с насадками (тефлоновые, не царапать!)
- Ножницы для PPR-труб
- Торцеватель (шейвер) — для армированных труб с фольгой
- Таймер (или просто секундомер телефона)

Время нагрева и фиксации зависит от DN — у каждого производителя есть таблица. Типичные значения для DN20: нагрев 5 с, соединение 4 с, охлаждение 2 мин.

> [!ОПАСНО]
> Насадки паяльника — 260°С. Не прикасайся к ним без рукавиц. Не оставляй паяльник включённым без надзора — нагрев продолжается. После работы — выключи и повесь на подставку, дай остыть.

### Опрессовочный насос

Для **гидравлических испытаний** смонтированной системы. Создаёт давление в трубопроводе выше рабочего — проверяет герметичность всех соединений.

- **Ручной опрессовочный насос** — простой, портативный. Создаёт давление до 60 бар. Достаточен для квартирных систем.
- **Электрический** — для больших объёмов или частого использования.

Нормативное испытательное давление для квартирного водопровода — **1.25 × рабочее давление**, но не менее 6 бар (СП 73.13330.2016).

## Группа 3: измерять и проверять

### Ключи

- **Трубный (газовый) ключ** — для затяжки резьбовых соединений на трубах. Есть три стандартных размера: №1 (до DN25), №2 (до DN50), №3 (до DN80). Для бытового монтажа хватает №1 и №2.
- **Разводной ключ** — универсальный, регулируемый. Менее удобен, чем специализированный, но полезен в нестандартных ситуациях.
- **Рожковые/накидные ключи** — для гаек компрессионных фитингов и накидных гаек (американок).

> [!СОВЕТ]
> Трубный ключ захватывает трубу зубьями и оставляет следы на хромированных и полированных поверхностях. Под ключ на красивую арматуру подкладывай мягкую прокладку или используй специальный ключ для смесителей (шестигранник или накидной).

### Рулетка и маркер

Разметка мест крепления, отмер длины труб, контроль расстояний между точками водоразбора. Рулетка 5 м — минимум. Маркер — для отметок на трубе.

### Строительный уровень

Для канализационных труб — контроль уклона (2 мм/м для DN50, 1 мм/м для DN110). Для горизонтальных водопроводных труб — чтобы подводки шли ровно.

Минимум: уровень-линейка 60 см. Лучше — пузырьковый уровень 80–100 см с делениями для контроля уклона.

### Манометр

Для контроля давления при опрессовке и в процессе эксплуатации. Диапазон для бытовых систем: 0–10 бар, точность 0.1 бар. Присоединяется к специальному гнезду в опрессовочном насосе или через тройник к системе.
